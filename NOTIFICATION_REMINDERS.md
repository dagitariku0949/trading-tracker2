## Troubleshooting

### Not Receiving Notifications?
1. Check browser notification permissions in settings
2. Make sure "Do Not Disturb" is off on your device
3. Ensure the website has notification permission
4. Try disabling and re-enabling in Settings

### Notifications Not Working on Mobile?
- iOS Safari: Requires iOS 16.4 or later
- Android Chrome: Should work by default
- Make sure the browser app has notification permissions
